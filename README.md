# Oddwork Prototype

Detta ar en klickbar multipage-prototyp for ny webbstruktur.

## Oppna lokalt

1. Ga till mappen `oddwork-prototype`.
2. Oppna `index.html` i webblasaren.
3. Klicka runt via header, knappar och interna lankar.

## Modulmarkering

Varje huvudmodul i prototypen ar markerad med:

- `Befintlig`
- `Anpassad`
- `Ny foreslagen`

Syftet ar att snabbt visa vad som kommer fran nuvarande webb, vad som omarbetas och vad som ar nytt forslag.
